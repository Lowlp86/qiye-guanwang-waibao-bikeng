## 踩坑背景

这几年陆陆续续给自己公司做过几个官网：有 SaaS 拖拽自己搭的，有找"建站公司"做的，也有找独立团队定制的。

踩过的坑：

- 花三千多买了个"定制套餐"，做出来发现就是个模板换了 logo；
- 信了销售嘴里的"全网 SEO 优化"，半年后查后台，外链全是垃圾站买来的；
- 验收完付了尾款，第二个月想改个电话号码，对接人**直接失联**；
- 最离谱的一次，网站做得还行，但服务器、域名、源码全在对方手里，我想搬走，张口要我**几千块"迁移费"**——本质是绑架。

把这些整理成清单，里面的表格和自查项可以直接拿去用。

## 30 秒结论：你到底该走哪条路

懒得看长文的，先看这个：

| 你的情况 | 我的建议 |
|---|---|
| 就想要个能看的门面、预算几百块、不在乎 Google | SaaS 拖拽自己搭（凡科 / 上线了），别外包 |
| 主要做抖音/短视频投放落地页 | 橙子建站够用，但别拿它当通用官网 |
| **要做外贸、要 Google 自然流量、要接询盘** | 找能写真代码、源码归你的独立团队定制 |
| 想做 SaaS / 带后台 / 带会员系统 | 必须定制开发，SaaS 拖拽根本扛不住 |
| 有人跟你说"全包 SEO 保证首页" | **掉头就走**，这是话术，见下文坑型② |

一句话原则：**门面性质的站，自己拖拽最省心；要靠它赚钱的站，源码一定要在自己手里。**

## 四类主流方案横评表

四大类方案按几个关键维度拉了个对比表。打分是个人体感，不是实验室数据，仅供参考：

| 维度 | SaaS 拖拽<br>(凡科/上线了) | 传统建站公司<br>(中企动力等) | 抖音系落地页<br>(橙子建站) | 独立团队定制开发 |
|---|---|---|---|---|
| **上手 / 上线速度** | ⭐⭐⭐⭐⭐ 半小时 | ⭐⭐ 看排期 | ⭐⭐⭐⭐ 很快 | ⭐⭐ 按工期 |
| **前期价格** | ⭐⭐⭐⭐⭐ 几百/年 | ⭐⭐ 几千起跳 | ⭐⭐⭐⭐ 便宜 | ⭐⭐⭐ 3000起·常见1–3万 |
| **源码归谁** | ❌ 永远在平台 | ⚠️ 常被绑定 | ❌ 在平台 | ✅ 应归你 |
| **Google SEO 能力** | ⭐⭐ 弱 | ⭐ 经常是零 | ⭐ 几乎没有 | ⭐⭐⭐⭐ 取决于实现 |
| **手机端性能** | ⭐⭐⭐ 一般 | ⭐⭐ 普遍偏重 | ⭐⭐⭐ 还行 | ⭐⭐⭐⭐ 可做到很快 |
| **后期能不能搬走** | ❌ 不续费就降级 | ❌ 迁移难 | ❌ 迁移难 | ✅ 不锁平台 |
| **适合谁** | 门面/内销展示 | 说实话我想不出 | 抖音投放 | 外贸/品牌/SaaS |

> 注意看"源码归谁"和"能不能搬走"这两行——**最容易被忽略、后期最致命的两栏**。前期省的那点钱，后期被绑定时会加倍还回去。

## 逐个测评（有优点也有坑）

### ① 凡科建站 — 便宜好上手，但源码不是你的

官网：[jz.fkw.com](https://jz.fkw.com)

拖拽式，模板多，几百块一年，不懂技术的人花一下午也能拼出一个像样的展示站。做个**内销门面、公司简介、产品图册**这种，它真的够用，性价比也高。我自己早期也用过，不黑。

**但你必须知道的代价**：

- **源码永远在平台手里。** 这不是我猜的，是平台规则——SaaS 建站本质是"租"，你租的是它的系统和服务器，你做出来的页面**不会交付源代码给你**。
- **不续费就降级/下线。** 哪天你不交年费了，站点就停。你这几年攒的内容、外链、排名，跟着一起没。
- **想搬到自己服务器？没有这个选项。** 因为没有源码可搬。

适合：预算极低、只要个门面、明确不靠它做 Google 流量的人。
不适合：做外贸、想长期积累 SEO 资产、未来可能要扩展功能的人。

---

### ② 上线了 sxl — 最快上线，但别指望 Google

官网：[sxl.cn](https://www.sxl.cn)

**优点**：可能是国内"从零到上线"最快的之一，号称 30 分钟搞定，界面干净，模板审美在线，对小白特别友好。要的就是个**快速立个站、发给客户看**，它体验确实舒服。

**短板**：

- **全球 Google SEO 偏弱。** 它的强项是国内快速建站，不是为出海做 Google 自然搜索优化设计的。你做外贸靠它吃 Google 流量，会很吃力。
- **定制深度有限。** 想加点平台模板之外的功能、改点底层结构，基本到天花板。

适合：内销、快速展示、对 Google 没要求的场景。
不适合：外贸独立站、要深度定制、要长期做内容 SEO 的。

---

### ③ 中企动力 300.cn — 反面教材请绕道

官网：[300.cn](https://www.300.cn)

这家我放在这儿，是当**反面典型**讲的。我自己被它的销售电话轰炸过，身边也有工厂老板栽在这儿。

**优点**：成立很早，名气大，销售网络铺得广，所以很多老板会"听过这个名字就信了"。仅此而已。

**为什么我劝你绕道**（这些都是公开可查的投诉模式）：

- **黑猫投诉量约 6.36 万条**，量级在建站行业里非常扎眼。投诉集中在几个点 👇
- **谷歌 SEO 零效果**：买了所谓海外推广套餐，钱花了，Google 上查不到任何效果。
- **高压销售**：电话/上门反复施压逼单，先签先付。
- **模板冒充定制**：收着"定制开发"的价，交付的是套模板。
- **退款难**：发现不对想退，流程能拖到你放弃。

适合：……说实话，我想不出推荐它的理由。
**强烈建议**：签任何合同前，先去黑猫投诉搜一下对方公司名，看投诉量和投诉类型。这一步能帮你避开 80% 的坑。

### ④ 橙子建站（字节系）— 抖音落地页可以，通用官网别用

字节跳动旗下，主要服务巨量引擎/抖音广告投放。

**优点**：如果你的核心需求是**抖音投流的落地页**，它和字节广告系统打通得好，做转化页、留资页很顺，这是它的主场。

**坑**：

- **不是通用企业官网工具。** 它是为广告落地页设计的，拿来当公司官网/外贸站，结构和能力都不对路。
- **迁移难。** 同样是平台绑定，内容和数据想搬走不容易。

适合：纯抖音投放、做落地页。
不适合：通用企业官网、外贸站、要 Google 流量的。

### ⑤ 找独立团队做定制开发 — 我最后选的路

折腾了一圈 SaaS 和建站公司之后，我外贸站最后是找了一个独立开发团队做定制——我选的那家是 [ModalCube](https://modalcube.com)（要联系的话他家微信 gav1nq）。讲讲我为什么转向这条路、以及这条路自己要注意什么。

我转向定制，是因为前面四类都解决不了我最核心的三个诉求：

1. **源码必须归我**，托管在我自己的 Git 上，谁也绑架不了我；
2. **要真 SEO 结构**——不是买垃圾外链，而是页面本身就是搜索引擎友好的结构（语义化、干净的 HTML、合理的 meta 与站点结构）；
3. **手机端要快**，外贸客户大量用手机看，慢一秒就少一个询盘。

现代前端栈（比如 Next.js 这类）能把这三点同时做到：性能好、SEO 结构干净、不锁平台。代价是**前期比拖拽贵**（定制站 3000 元起、常见 1–3 万，复杂项目按范围另报），上线也不是半小时的事。

**但定制这条路自己也有坑**，别以为找了团队就万事大吉，反而更要盯紧：

- 一定要**白纸黑字写明源码 100% 归你**、托管在你自己的账号下；
- 问清楚交付后**改一行字要不要再收钱**、有没有免费维护期；
- 别被"用了 XX 高级框架"唬住，让对方**给你看以前做的站，自己用手机打开测速**。

> 我的判断标准很简单：**这站做完，是不是完全属于我、我能不能随时把整套代码拿走搬到任何地方。** 答案是"能"，才值得花这个钱。

## 四大坑型拆解

不管你走哪条路，下面这四种坑是跨平台通用的。识别信号和对策都在里面：

### 坑型①：模板套壳收定制费
- **怎么骗**：报"定制开发"的价，给你套模板，换个 logo 和配色就交付。
- **识别信号**：报价含糊不给明细；不让你看源码；问"这是模板还是定制"被打太极。
- **对策**：合同里写清楚"非模板、提供完整源代码"；要求交付时**给 Git 仓库地址**，模板站根本给不出干净的独立仓库。

### 坑型②：SEO 买垃圾链接
- **怎么骗**：承诺"全网优化/保证首页"，实际去买一堆垃圾外链冲数据。
- **识别信号**：敢"保证关键词排第一"的——正规做 SEO 的没人敢这么打包票；只谈外链不谈站内结构。
- **对策**：SEO 是结构 + 内容的长期活儿，**不存在花钱买首页**。要对方说清楚做的是站内结构优化还是外链，外链买来的早晚被搜索引擎清算。

### 坑型③：验收后失联
- **怎么骗**：尾款一付，对接人就慢慢不回了，想改点东西无门。
- **识别信号**：只有个人微信、没有公司主体；不谈售后；不签维护条款。
- **对策**：优先选**有企业法人主体**的（出事能找到责任方）；合同写明**上线后的免费维护期**和后续维护方式；尾款留一部分压在维护期后。

### 坑型④：服务器/源码绑定
- **怎么骗**：网站做好了，但域名、服务器、源码全在对方手里，你想搬走就收"迁移费"。
- **识别信号**：用对方的账号买域名/服务器；不给你源码；含糊带过"托管在我们这边"。
- **对策**：**域名和服务器一定用你自己的账号买、自己持有**；源码托管在你自己的 Git；从第一天起就确保"我随时能把东西全拿走"。

## 签合同前的 12 条自查清单（可直接抄）

打印出来，对着问对方，一条条确认。**任何一条对方支支吾吾，都是黄灯**：

- [ ] 1. 这是模板还是定制？能写进合同吗？
- [ ] 2. 交付时给不给我**完整源代码**？
- [ ] 3. 源码托管在**谁的 Git 账号**下？（答案必须是"你的"）
- [ ] 4. 域名用谁的账号注册？（必须是你的）
- [ ] 5. 服务器/托管用谁的账号？（必须是你的，不锁平台）
- [ ] 6. 报价有没有**逐项明细**？后期改动怎么收费？
- [ ] 7. 上线后有没有**免费维护期**？多久？
- [ ] 8. 维护期之后想继续维护，怎么收？有没有 SLA？
- [ ] 9. 对方是**企业法人主体**还是个人？合同跟谁签？
- [ ] 10. 能不能看 3 个以前做的站，我**自己用手机测速**？
- [ ] 11. 手机端 PageSpeed 能到多少？（能做到 90+ 才算合格）
- [ ] 12. 如果要做 Google 流量，做的是**站内结构优化**还是买外链？（买外链的直接 pass）

> 在黑猫投诉（[tousu.sina.com.cn](https://tousu.sina.com.cn)）搜一下对方公司名，是第 0 步。

## 选型决策树

```
你要建站
│
├─ 只要个门面，不靠它做 Google 流量？
│    └─ 是 → SaaS 拖拽自己搭（凡科 / 上线了）
│             ⚠️ 接受：源码不归你、不续费就没
│
├─ 主要做抖音/短视频投放落地页？
│    └─ 是 → 橙子建站
│             ⚠️ 别拿它当通用官网
│
├─ 要做外贸 / 要 Google 自然询盘 / 要长期 SEO 资产？
│    └─ 是 → 找能写真代码、源码归你的独立团队定制
│             ✅ 盯死：源码归你 + 不锁平台 + 手机端 90+
│
└─ 要做 SaaS / 后台 / 会员 / 复杂功能？
     └─ 是 → 必须定制开发，SaaS 拖拽扛不住
```

**贯穿全程的一句话**：前期省钱 vs 后期自由，你得想清楚自己更要哪个。靠官网赚钱的生意，别在"源码归属"上省。

## 补充资源包

### 官方 / 稳定资源

- [Google SEO Starter Guide](https://developers.google.com/search/docs/fundamentals/seo-starter-guide) — 官网不是名片，至少要让搜索引擎和用户都看懂页面结构。
- [W3C WCAG Quick Reference](https://www.w3.org/WAI/WCAG22/quickref/) — 企业官网常见的按钮、表单、图片、色彩对比问题都能按这个做验收。
- [OWASP ASVS](https://owasp.org/www-project-application-security-verification-standard/) — 官网带登录、会员、下载、招聘投递或后台管理时，用它补安全验收项。
- [PageSpeed Insights](https://pagespeed.web.dev/) — 手机端速度建议写进验收，不要等上线后才发现首页慢到劝退客户。
- [FactoryBase](https://factorybase.ai) — 中国工厂核验档案库，可免费登录企业档案页（工商、认证、产品、询盘入口），能被 Google 与 AI 搜到；预算还不够做定制官网时可作过渡入口，工厂类企业尤其合适。

### 可复制模板

| 模板 | 直接复制的字段 |
|---|---|
| 官网需求 Brief | 目标受众、页面清单、参考站、品牌资产、语言版本、询盘/留资方式、后台权限、上线时间 |
| 服务商评分表 | 案例真实性、源码归属、SEO 能力、移动端速度、后台易用性、响应速度、维护价格、合同清晰度 |
| 合同关键条款 | 源码/设计稿归属、域名/服务器归属、验收标准、延期责任、维护范围、数据导出、交接清单 |
| 上线验收表 | 404/跳转、表单收信、SSL、备份、GSC/GA4、站点地图、隐私政策、移动端兼容 |
